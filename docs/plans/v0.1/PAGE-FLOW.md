# 数据脱敏工具 v0.1 页面流程与交互设计

> 基于 USER-STORIES.md 中 20 个核心用户故事，定义页面结构、流转关系和交互细节。

## 一、设计原则

- **单窗口视图切换**：用户心理模型是"丢文件进去，拿安全文件出来"，流程线性引导，不给决策负担
- **简洁专业风格**：白底为主，淡灰分割，主色调蓝/绿系（传递"安全感"），圆角卡片，体现工具可靠性
- **各文件类型原生展示**：Excel 用表格渲染、Word 用段落渲染，保留用户熟悉的格式观感
- **面向非技术用户**：ToC 体验标准，交互直觉自然，无需学习成本

## 二、页面结构总览

### 视图（5个）

| 视图 | 说明 | 入口 |
|------|------|------|
| P1 首页 | 文件导入区 + 最近脱敏任务（条件展示） | 应用启动 |
| P2 识别预览 | 文件内容高亮 + 汇总条 + 手动修正 | P1 导入文件后 |
| P3 脱敏结果 | 左右并排对比 + 导出 | P2 执行脱敏后 |
| P4 历史任务 | 任务列表 + 还原/删除操作 | 顶部栏入口 |
| P5 还原结果 | 左右并排对比 + 导出 | P4 或 P1 点击还原 |

### 弹窗/面板（3个）

| 组件 | 形式 | 说明 |
|------|------|------|
| M1 词典管理 | 右侧抽屉 | 任何页面可开启，添加/编辑/删除自定义词条 |
| M2 策略配置 | 右侧面板 | P2 页面专属，按类型配置脱敏策略 |
| M3 敏感项详情 | 浮层 | P2 页面点击高亮项弹出 |

## 四、各页面详细设计

### P1 首页 — 文件导入

**对应用户故事**: US-01, US-02, US-03, US-18（最近任务）

#### 布局

```
┌─────────────────────────────────────────────┐
│  脱敏工具                [词典管理] [历史任务] │
├─────────────────────────────────────────────┤
│                                             │
│         ┌─────────────────────────┐         │
│         │                         │         │
│         │     拖拽文件到这里        │         │
│         │                         │         │
│         │    或 [点击选择文件]      │         │
│         │                         │         │
│         │   支持 Excel/CSV/Word    │         │
│         │   单个文件，最大50MB      │         │
│         └─────────────────────────┘         │
│                                             │
│  最近脱敏任务               [查看全部 >]     │
│  ┌─────────────────────────────────────┐    │
│  │ 合同_甲方.docx    1月15日   28处     │    │
│  │                      [还原] [详情]   │    │
│  ├─────────────────────────────────────┤    │
│  │ 客户信息表.xlsx   1月14日   156处    │    │
│  │                      [还原] [详情]   │    │
│  └─────────────────────────────────────┘    │
│                                             │
└─────────────────────────────────────────────┘
```

- **首次使用**：无历史任务时，"最近脱敏任务"区域不显示，首页只有导入区，干净纯粹
- **有历史任务后**：导入区下方自然出现最近 2-3 条任务卡片
- **"查看全部"**：跳转 P4 历史任务列表
- **卡片上的 [还原]**：直接弹出文件选择对话框 → 选择 AI 结果文件 → 进入 P5

#### 交互细节

| 操作 | 行为 |
|------|------|
| 拖拽文件进入窗口 | 拖入区域边框高亮变色，提示"松开导入" |
| 松开文件 | 校验格式和大小 → 通过则进入解析 → 跳转 P2 |
| 点击"选择文件" | 打开系统文件对话框，过滤器只显示支持格式 |
| 格式不支持 | Toast 提示"目前支持 Excel(.xlsx/.xls)、CSV(.csv/.tsv)、Word(.docx)" |
| 文件过大 | Toast 提示"文件过大，请拆分后再试（最大 50MB）" |
| 文件损坏 | Toast 提示"文件无法读取，请检查文件是否完好" |
| 密码保护 | Toast 提示"文件有密码保护，请先移除密码" |
| 解析中 | 导入区替换为加载动画 + "正在解析文件..." |

**对应用户故事**: US-04, US-05, US-06, US-07, US-08, US-09, US-10, US-11, US-12

#### 布局

```

├─────────────────────────────────────────────────────────┤
│  共识别 28 处 │ 手机号(12) 身份证(5) 人名(8) 机构名(3) │ NER识别中...◌ │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  文件内容区（全宽）                                       │
│                                                         │
│  ┌─────────────────────────────────────────────────┐    │
│  │ [张三] 的手机号是 [13812345678]，                  │    │
│  │ 身份证号 [110101199001011234]                     │    │
│  │ ...                                             │    │
│  │ （Excel 以表格形式，Word 以段落形式）               │    │
│  └─────────────────────────────────────────────────┘    │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  文件: 合同.docx | 大小: 1.2MB               [开始脱敏]  │
└─────────────────────────────────────────────────────────┘
```

#### 顶部汇总条

- 最左侧：总数 "共识别 X 处"
- 中间：各类型标签 + 数量，只展示有识别结果的类型
- 每个标签前带复选框，控制该类型全选/取消
- 点击标签文字可筛选/定位到该类型的第一个高亮项
- 最右侧：NER 识别状态（"NER识别中..." 完成后消失）

#### 内容区交互

| 操作 | 行为 |
|------|------|
| 文件加载完成 | 立即触发规则引擎 + 词典匹配，毫秒级返回，高亮标记 |
| 规则结果展示后 | 自动触发 NER 异步识别，汇总条显示"NER识别中..." |
| NER 完成 | 新识别项追加高亮，汇总条数字更新，状态指示器消失 |
| 点击高亮区域 | 弹出 M3 详情浮层 |
| 选中未标记文本 | 选区上方出现浮动工具条，可选择标记为哪种敏感类型 |
| Excel/CSV | 表格形式渲染，高亮在单元格内文本上 |
| Word | 段落形式渲染，高亮在行内文本上 |

#### M3 敏感项详情浮层

点击高亮区域时弹出：

```
┌──────────────────────────┐
│  原始文本: 13812345678    │
│  识别类型: 手机号         │

│                          │
│  脱敏策略:               │
│  ● 掩码  ○ 替换  ○ 泛化  │
│  预览: 138****5678       │
│                          │
│  [取消标记]    [确认]     │
└──────────────────────────┘
```

| 操作 | 行为 |
|------|------|
| 切换策略 | "预览"行实时更新脱敏效果 |
| 取消标记 | 高亮消失，汇总条数字减少，浮层关闭 |
| 确认 | 应用该项的个性化策略，浮层关闭 |
| 点击浮层外部 | 浮层关闭，不做修改 |

## 七、前端状态流转

```
idle ──(导入文件)──→ parsing ──(解析完成)──→ detecting
                        │                      │
                     (解析失败)            (规则完成，NER进行中)
                        │                      │
                        ▼                      ▼
                      error              previewing ──(NER完成)──→ previewing(完整)
                                               │
                                          (开始脱敏)
                                               │
                                               ▼
                                         desensitizing ──(完成)──→ result
                                                                     │
                                                                (导出/新文件)
                                                                     │
                                                                  idle/P1

── 还原分支 ──
idle ──(选择历史任务+导入文件)──→ restoring ──(完成)──→ restored
```
